# FarmerNote1 (初芽巡田) 官网前端工程

这里是“初芽巡田”的官方宣传网站、用户服务协议和隐私政策的静态页面目录。

## 🌐 域名规划与结构说明

在目前的开发与提交流程中，为了兼顾合规审查（ICP备案）和敏捷开发，我们的外部网络结构设计如下：

*   **`https://api-test.chuya.wang`**: 内部专用。只用于小程序和 Flutter 请求数据库数据（BaaS），属于**静默请求网关**，不在浏览器端直接展示页面内容。
*   **`https://web-t.chuya.wang`**: 对外门面。专门用于承载当前 `website` 目录下的 HTML 文件。如果需要修改隐私弹窗里的跳转链接，**请务必确保链接指向此域名**以免误导用户或触发 404！

## 🚀 如何快速部署网站更新

当你在本地修改了 `site.js`、`privacy.html`，或是首页增加了一张图，想要把修改发布给线上的所有人看时，可以采用以下极简的一键部署方案。

### 使用方法

1.  打开你的 Mac 终端（Terminal）或是在 VSCode 里的终端。
2.  确保进入了工程**根目录 (`/Users/wzp/Documents/GitHub/FarmerNote1`)**。
3.  执行下面的发布指令：

```bash
./scripts/deploy_website.sh
```

### 命令底层说明

当你敲下回车后，脚本会自动帮你执行以下工作：
1.  自动把 `website/` 文件夹**打包压缩**。
2.  通过免密 **SSH 隧道传输**到新加坡测试服务器 (`43.119.88.83`)。
3.  指挥远端服务器解压，并**精准覆盖**掉上一版的旧文件。
4.  自动打扫战场（清理临时产生的压缩包）。

只要你看到终端输出 `🎉 线上门面已更新: https://web-t.chuya.wang/` 的撒花提示，直接去浏览器里强制刷新页面（`Cmd+Shift+R`），你的最新才华就呈现在网上了！
