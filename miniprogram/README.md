# FarmerNote 微信小程序版

这个目录是把当前仓库的核心功能改写成原生微信小程序后的实现。

## 包含的功能

- `记录`：写文字巡田记录，可选挂一个提醒时间
- `时间线`：按创建时间倒序展示所有记录，并显示对应任务状态
- `待办`：按 `即将到来 / 已逾期 / 已完成` 分组，可完成、改时间、删除任务
- `本地持久化`：通过 `wx.setStorageSync` 保存在当前微信客户端本地

## 和原生 App 的差异

- 微信小程序不能像原生 App 那样做离线本地闹钟
- 这里会在保存未来任务时自动尝试写入手机系统日历；如果当前微信环境或权限不支持，仍然保留小程序内任务管理

## 打开方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录指向 `/Users/wzp/Documents/GitHub/FarmerNote1/miniprogram`
4. `AppID` 可先用测试号或游客模式
5. 导入后直接预览或真机调试

## 目录说明

- `app.*`：小程序入口和全局样式
- `pages/record`：记录页
- `pages/timeline`：时间线页
- `pages/tasks`：待办页
- `utils/store.js`：本地存储和业务操作
- `utils/date.js`：日期处理和格式化
